# 暗区突围无限 - 装备随机生成器

## 项目背景

暗区突围无限（AQTWWX）装备随机生成器是一个基于 Web 的工具，用于帮助玩家随机生成符合特定战备值区间的装备搭配。该工具基于游戏内的真实装备数据和搭配规则。

## 核心功能

### 1. 模式选择

三种生成模式，每种模式对应不同的战备值区间：

- **普通模式**：战备值 < 450,000
- **封锁模式**：150,000 ≤ 战备值 ≤ 1,000,000
- **强化封锁模式**：战备值 ≥ 450,000

### 2. 装备类别与设置

| 类别 | 说明 | 可选模式 |
|------|------|----------|
| 防弹衣/弹挂 | 互斥选择（有甲弹挂 或 护甲+无甲弹挂） | 随机/指定 |
| 头盔 | 含普通头盔和自闭头 | 随机/指定 |
| 面罩 | 需与头盔兼容 | 随机/指定 |
| 背包 | 各类携行背包 | 随机/指定 |
| 耳机 | 战术耳机 | 随机/指定 |
| 防毒面具 | 防护面具 | 随机/指定 |
| 枪械 | 突击步枪、冲锋枪、霰弹枪等 | 随机/指定 |

### 3. 装备搭配规则

#### 冲突规则
1. **防毒面具冲突**
   - 与面罩不能同时装备
   - 与特定头盔（56K型直升机头盔、DOD9型防爆头盔）不能同时装备

2. **耳机冲突**
   - 自闭头不能搭配耳机

3. **面罩兼容性**
   - 每个头盔只能搭配特定的面罩（见头盔 `available_face_shields` 属性）

4. **防弹衣/弹挂互斥**
   - 使用有甲弹挂时不能装备其他护甲或弹挂

#### 自闭头列表
摩托车头盔、坦克兵防护帽、飞行员头盔、安保防爆头盔、SH12军用头盔、56K型直升机头盔、SH40军用头盔、SH18军用头盔、AN95重型防爆头盔、SH马斯卡2型头盔、SH60军用头盔、SH50军用头盔、RSP重装战术头盔、6BNT型头盔、RST特种部队头盔、HG84特攻型头盔、SH65军用头盔、DOD9型防爆头盔

## 使用方式

1. 在浏览器中打开 `index.html`
2. 选择生成模式（普通/封锁/强化封锁）
3. 勾选需要生成的装备类别
4. 每个类别可选择"随机"或"指定"
5. 点击"随机生成"按钮

## 技术实现

- **纯前端**：独立的 HTML 文件，无需服务器
- **响应式**：支持桌面端和移动端
- **动画效果**：装备卡片滑动淡入显示
- **冲突检测**：实时检测并提示装备冲突

## 开发注意事项

1. **价格使用整数**，不涉及小数
2. **枪械只取基础枪械**，不涉及配件
3. **最多尝试 50 次生成**，失败时提示用户调整配置
4. **数据文件中的 `#` 号前缀**是文件命名的一部分

## 数据源文件

- `AQ/# 暗区突围无限_装备数据库.txt` - 头盔、面罩、防弹衣、弹挂、背包、耳机、防毒面具数据
- `AQ/# 暗区突围无限_枪械数据库.txt` - 各类枪械数据
- `AQ/# 装备搭配规则.txt` - 装备搭配和冲突规则
